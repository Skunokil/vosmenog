# Vosmenog — фреймворк дисциплинированного агента для opencode

Восилий (Vosmenog) — режим строгого исполнения для [opencode](https://opencode.ai):
агент-исполнитель по детальному ТЗ, который **не уходит в самодеятельность**.
Не архитектор, не трогает прод, документирует работу эпиками (DoR/DoD/AC), эскалирует
развилки вместо того, чтобы решать их догадкой.

## Что внутри

- **METHOD.md** — ядро метода (always-context).
- **agents/Vosmenog.md** — таб-режим с урезанными правами (барьер периметра).
- **EPIC / TASK / BUG** — шаблоны документирования.
- **STARTUP.md** — протокол памяти между сессиями.
- **setup.sh** — установщик (раскладка, конфиг, периметр, лаунчер).

## Установка

```bash
git clone <этот-репозиторий> vosmenog
cd vosmenog
chmod +x setup.sh
./setup.sh
```

Подробно — `GUIDE.md`. Для установки с ИИ-наставником — кинь `INSTALL_PROMPT.md`
первым сообщением в чат с ассистентом, приложив папку.

## Зачем

Метод-как-текст модель переезжает под нагрузкой. Vosmenog ставит барьеры структурой:
урезанные права (прод физически недостижим), обязательный TASK перед работой,
развилки эскалируются. Дисциплина — конфигом, не увещеванием.
